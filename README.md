# MR Data Analysis

A part of [MathRocks](https://github.com/MathRocks/MathRocks).

## Contents

* `+Plot` — routines for plotting cycles and extrema in a data set.
* `+Print` — routines for comparing data structures.
* `+Utils` — various helper functions.
